El Springfield Hospital ha solicitado al departamento de Tecnología; la creación de un sistema 
para gestionar turno de asistencia a los pacientes. Para agilizar el proceso de los datos de los 
pacientes, se le ha permitido utilizar la información de los mismo a través de una API que van a 
consultar. Se debe crear interfaces en donde los actores tengan participación de acuerdo con su 
caso de uso.
